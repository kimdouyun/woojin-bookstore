# 📚 온라인 책 리뷰 사이트

나만의 온라인 책 리뷰 사이트입니다. 읽은 책들의 리뷰를 작성하고, 다른 사람들과 공감 댓글을 나눌 수 있습니다.

## 주요 기능

- 🏠 **랜딩 페이지**: 입장 버튼이 있는 아름다운 홈페이지
- 📖 **책 목록**: 읽은 책들의 표지 이미지를 그리드 형태로 표시
- 📝 **책 상세 페이지**: 각 책의 상세 리뷰 확인
- 💬 **댓글 기능**: 리뷰에 공감 댓글 작성
- 🔐 **회원가입/로그인**: 사용자 인증 시스템
- 🔧 **관리자 기능**: 관리자만 책 추가, 수정, 삭제 가능
- 👥 **사용자 관리**: 관리자가 다른 사용자에게 관리자 권한 부여

## 기술 스택

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React**

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 초기 관리자 계정 설정

관리자 계정을 먼저 생성해야 합니다:

```bash
node scripts/setup-admin.js <사용자명> <비밀번호>
```

예시:
```bash
node scripts/setup-admin.js admin mypassword123
```

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 사용 방법

### 일반 사용자
1. **홈페이지**: 처음 접속하면 입장 버튼이 있는 랜딩 페이지가 표시됩니다.
2. **책 목록**: 입장 버튼을 클릭하면 읽은 책들의 목록이 표시됩니다.
3. **책 상세**: 책을 클릭하면 리뷰를 확인하고 댓글을 작성할 수 있습니다.

### 관리자
1. **로그인**: `/login` 페이지에서 관리자 계정으로 로그인
2. **관리자 페이지**: 로그인 후 관리자 페이지에서 책을 추가/수정/삭제할 수 있습니다.
3. **사용자 관리**: 관리자 페이지의 "사용자 관리" 버튼을 클릭하여 다른 사용자에게 관리자 권한을 부여할 수 있습니다.

## 인증 시스템

### 회원가입
- `/login` 페이지에서 "회원가입" 모드로 전환
- 사용자명, 이메일(선택), 비밀번호 입력
- 비밀번호는 최소 6자 이상이어야 합니다.

### 로그인
- 관리자 권한이 있는 계정으로만 관리자 페이지에 접근 가능
- 일반 사용자는 댓글 작성만 가능

### 관리자 권한 설정
1. 관리자로 로그인
2. 관리자 페이지에서 "사용자 관리" 버튼 클릭
3. 사용자 목록에서 "관리자 지정" 버튼 클릭

## 데이터 저장

데이터는 `data/` 폴더에 JSON 파일로 저장됩니다:
- `books.json`: 책 정보
- `comments.json`: 댓글 정보
- `users.json`: 사용자 정보
- `sessions.json`: 세션 정보

## 보안 참고사항

⚠️ **중요**: 현재 구현은 개발/프로토타입용입니다. 프로덕션 환경에서는 다음을 권장합니다:

1. **비밀번호 해싱**: 현재는 SHA-256을 사용하지만, 프로덕션에서는 `bcrypt` 사용 권장
2. **세션 관리**: Redis나 데이터베이스 사용 권장
3. **HTTPS**: 프로덕션에서는 반드시 HTTPS 사용
4. **환경 변수**: 민감한 정보는 환경 변수로 관리
